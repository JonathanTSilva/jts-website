# Now Template

Required metadata:
- title
- language
- updatedAt

Optional metadata:
- summary
- translationKey
- status label

## Frontmatter Example

```markdown
---
title: "What I'm doing now"
language: "en"
updatedAt: "2026-03-19"
summary: "Current focus on embedded systems and personal projects."
translationKey: "now-index"
status: "Active"
---
```

## Required Sections
The now page should present a concise snapshot of current focus, work, learning, and active directions.
